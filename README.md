# lita-gimli

Lita handler that contributes an axe.

## Installation

Add lita-gimli to your Lita instance's Gemfile:

```ruby
gem "lita-gimli"
```

## Usage

```
geoffrey: and you'll have my sword...
killpack: and my bow!
Lita: AND MY AXE!
```
